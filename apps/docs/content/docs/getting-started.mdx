Get up and running with WallpaperDB in under 10 minutes.

## Prerequisites

- Docker & Docker Compose
- Node.js 22+
- pnpm 8+
- Make

See [Local Setup](/docs/infrastructure/local-setup) for installation instructions.

## Quick Start

```bash
# 1. Clone and install
git clone https://github.com/your-org/wallpaperdb.git
cd wallpaperdb
pnpm install

# 2. Start infrastructure
make infra-start

# 3. Run migrations
pnpm --filter @wallpaperdb/ingestor db:migrate

# 4. Start services
make dev
```

**Access points:**
- Gateway (GraphQL): http://localhost:3000
- Ingestor: http://localhost:3001
- Media: http://localhost:3002
- Variant Generator: http://localhost:3004
- Web Frontend: http://localhost:3005
- Grafana: http://localhost:3000 (admin/admin)
- MinIO Console: http://localhost:9001 (minioadmin/minioadmin)

## Test the Upload Flow

```bash
# Create test image
curl -o test.jpg https://via.placeholder.com/1920x1080.jpg

# Upload wallpaper
curl -X POST http://localhost:3001/upload \
  -F "file=@test.jpg" \
  -F "userId=test_user"

# Response includes wallpaper ID
# {"wallpaperId": "wlpr_...", "uploadState": "processing"}

# Retrieve wallpaper (use ID from above)
curl http://localhost:3002/wallpapers/wlpr_... > retrieved.jpg

# Resize on-the-fly
curl "http://localhost:3002/wallpapers/wlpr_...?w=800&h=600&fit=contain" > resized.jpg

# Query via GraphQL
curl -X POST http://localhost:3000/graphql \
  -H "Content-Type: application/json" \
  -d '{"query": "{ searchWallpapers(first: 10) { edges { node { wallpaperId } } } }"}'
```
