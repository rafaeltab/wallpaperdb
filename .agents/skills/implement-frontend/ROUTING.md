# Routing

TanStack Router with file-based routing via the Vite plugin. Routes live in `apps/web/src/routes/`.

> **Never edit `src/routeTree.gen.ts`** — it is auto-generated by the Vite plugin on every save. Edits will be overwritten.

## Adding a New Route

1. Create a file in `src/routes/` following the naming convention
2. Save the file — the Vite plugin regenerates `routeTree.gen.ts` automatically
3. The route is immediately available; no manual registration needed

## File Naming Convention

| File | URL |
|---|---|
| `src/routes/index.tsx` | `/` |
| `src/routes/upload.tsx` | `/upload` |
| `src/routes/wallpapers.$wallpaperId.tsx` | `/wallpapers/:wallpaperId` |
| `src/routes/settings/index.tsx` | `/settings` |
| `src/routes/__root.tsx` | Root layout (wraps all routes) |

- Use `.` as a path segment separator for nested paths
- Prefix a segment with `$` for dynamic params: `$paramName`
- `__root.tsx` is the layout that wraps everything (header, sidebar, `<Outlet />`)

## Basic Route File

```tsx
import { createFileRoute } from '@tanstack/react-router'

export const Route = createFileRoute('/my-page')({
  component: MyPageComponent,
})

function MyPageComponent() {
  return <div>My Page</div>
}
```

## Route with Dynamic Param

```tsx
import { createFileRoute } from '@tanstack/react-router'

export const Route = createFileRoute('/wallpapers/$wallpaperId')({
  component: WallpaperDetailPage,
})

function WallpaperDetailPage() {
  const { wallpaperId } = Route.useParams()
  return <div>Wallpaper: {wallpaperId}</div>
}
```

## Navigation

```tsx
import { Link, useNavigate } from '@tanstack/react-router'

// Declarative
<Link to="/upload">Upload</Link>
<Link to="/wallpapers/$wallpaperId" params={{ wallpaperId: id }}>View</Link>

// Imperative
const navigate = useNavigate()
navigate({ to: '/upload' })
```

## Search Params

```tsx
export const Route = createFileRoute('/search')({
  validateSearch: (search) => ({
    query: (search.query as string) ?? '',
  }),
  component: SearchPage,
})

function SearchPage() {
  const { query } = Route.useSearch()
  // ...
}
```

## Lazy Loading

For code-splitting, use `createLazyFileRoute` instead of `createFileRoute`:

```tsx
import { createLazyFileRoute } from '@tanstack/react-router'

export const Route = createLazyFileRoute('/heavy-page')({
  component: HeavyPageComponent,
})
```
